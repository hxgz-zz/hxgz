# 和谐国战方案 0.1版

## 1. 顺时针建立匪线
## 2. 国战前45分钟逆时针任意刷城,后15分钟各自收回属地。
## 3. 前45分钟不允许卡城,挂小号以及互相攻击（约炮的除外）,后15分钟可以卡城保护属地。
## 4. 奶号可以挂上收金。
